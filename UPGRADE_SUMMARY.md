   - **Code Change**:

     ```typescript
     // Before (Next.js 15)
     return new Response(data, { ... });

     // After (Next.js 16)
     const uint8Array = new Uint8Array(buffer);
     const blob = new Blob([uint8Array], { type: 'application/zip' });
     return new NextResponse(blob, { ... });
     ```
